fixi project demo

Small Node-backed demos exercising the all-in-one bundle (fixi + moxi + ssexi + paxi + rexi) against real server endpoints.

  • 📒 Contacts CRUD
    Searchable address book with inline editing. Active search via debounced moxi + custom fx-trigger; new contact POST, save PUT, remove DELETE through fixi; paxi morph keeps the search input focused while the list updates per keystroke.
  • 📋 TodoMVC
    Classic todo app: per-row toggle + delete, toggle-all, clear-completed, all/active/completed filters. paxi morph-swaps keep the list stable across every action; moxi drives the toggle-all indicator, the footer count, and the filter bar; server templates are eta.
  • 🤖 Gibberish bot
    A fake LLM that streams word-by-word nonsense via SSE. fixi sends the question as a POST that returns text/event-stream; ssexi appends each token; moxi manages the conversation log.
  • 💬 Chat
    Shared chat room. ssexi streams messages in over SSE; fixi posts the compose form; moxi resets and refocuses after send. Open two tabs.
  • 📊 Live dashboard
    One ssexi SSE stream, three panels, three different swap modes. Each message's event: field is JSON like {"target":"#clock"} so the server fans the same stream out to a clock (innerHTML), a metrics block (innerHTML), and an activity log (beforeend) without any extra connections.
  • 🐍 Snake
    Server runs the game loop and ticks state on a timer. ssexi streams each tick out to #board, #score, and #status through the same connection; rexi posts every direction change as a dir=... form; moxi wires the arrow keys at on-init. Open two tabs and fight over the snake.

Source: bigskysoftware/fixi-project. Start with npm run demo from the project root.
